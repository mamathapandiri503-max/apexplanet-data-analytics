# ApexPlanet Data Analytics - Task 1

## Project Overview
This repository contains my Task 1 submission for the ApexPlanet Data Analytics Internship.

## Dataset
- Online Retail (E-commerce Sales) Dataset

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Task Completed
### Data Cleaning
- Loaded dataset into Pandas
- Handled missing values
- Removed duplicates
- Fixed data types
- Removed outliers using the IQR method
- Saved the cleaned dataset

### Exploratory Data Analysis (EDA)
- Generated statistical summary
- Created histograms
- Created boxplots
- Created bar charts
- Generated a correlation heatmap
- Analyzed monthly sales trends using a line chart

- Most orders contained small quantities.

- Monthly sales showed noticeable trends.
- Quantity sold positively influenced total sales.
